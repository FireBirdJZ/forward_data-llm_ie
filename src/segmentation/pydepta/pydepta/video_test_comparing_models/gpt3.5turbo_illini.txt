To solve this task, we can follow these steps:

1. Split the given text into a list of professor details using the delimiter " ".
2. Iterate through each professor detail and create a dictionary for each professor.
3. Extract the name, position, and research interests (if present) from each professor detail.
4. Create a JSON object for each professor using the extracted information.
5. Append each JSON object to a list.
6. Convert the list of JSON objects into a JSON array.

Here's the solution in Python:

```python
import json

# Given text with professor details
text = "Michael Twidale Professor, Information Sciences Luther Tychonievich he/him/his Teaching Associate Professor Lav R. Varshney Associate Professor, Electrical and Computer Engineering"

# Split the text into professor details using delimiter " "
professor_details = text.split(" ")

# Create a list to store the JSON objects
json_objects = []

# Iterate through each professor detail
for i in range(0, len(professor_details), 3):
    # Extract name, position, and research interests (if present)
    name = professor_details[i]
    position = professor_details[i+1]

    # Check if research interests are present
    if i+2 < len(professor_details):
        research_interests = professor_details[i+2]
    else:
        research_interests = None

    # Create a dictionary for the professor
    professor_dict = {
        "name": name,
        "position": position,
        "research_interests": research_interests
    }

    # Append the dictionary to the list
    json_objects.append(professor_dict)

# Convert the list of JSON objects into a JSON array
json_array = json.dumps(json_objects)

print(json_array)
```
